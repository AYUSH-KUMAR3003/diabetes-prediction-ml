# 🩺 Diabetes Prediction using Machine Learning

## 📌 About the Project
In this project, I developed a machine learning model to predict diabetes using patient medical data.

## 🎯 Objective
To classify whether a person is diabetic or non-diabetic based on health parameters.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Seaborn & Matplotlib
- Scikit-learn

## 🤖 Model Used
- K-Nearest Neighbors (KNN)

## 📊 Results
The model achieved an accuracy of around 77%.

## 🔍 Key Steps
- Data Cleaning (handling missing values)
- Exploratory Data Analysis (EDA)
- Feature Scaling
- Model Training and Testing

## 📁 Dataset
The dataset contains features like Glucose, Blood Pressure, BMI, Age, etc.
